D-A-CH Email Suggestion

Typo correction for email inputs

New API recommended

1. <EmailSuggestion> — controlled, bring your own input
2. useEmailSuggestion hook — custom rendering, submit-guard via getCorrected()
import { EmailSuggestion, useEmailSuggestion, DACH_DOMAINS } from 'dach-email-suggestion';

// 1. Render component
const [email, setEmail] = useState('');
<input value={email} onChange={e => setEmail(e.target.value)} />
<EmailSuggestion email={email} onAccept={setEmail} />

// 2. Headless hook — synchronous submit check
const { suggestion, getCorrected } = useEmailSuggestion(email);
if (getCorrected()) askUserBeforeSubmitting();

Legacy API compatibility

3. <DACHSuggestions id> — backwards-compatible data-* wiring
4. Legacy with callbacks + extended domain list (adds company.de)
import { DACHSuggestions, DACH_DOMAINS } from 'dach-email-suggestion';

<input type="email" data-dach-suggestion="mailCheck" />
<DACHSuggestions id="mailCheck"
  domains={[...DACH_DOMAINS, 'company.de']}
  onSuggest={track}
/>
